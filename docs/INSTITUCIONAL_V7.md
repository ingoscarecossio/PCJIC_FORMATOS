# Versión Institucional 7.0.0

## Propósito

Elevar la suite FD-GC71/FD-GC72 de un sistema de diligenciamiento a una plataforma de gestión académica documental con gobierno, coherencia, aprobación y trazabilidad.

## Módulos nuevos

1. **Banco institucional de asignaturas**: catálogo maestro de asignaturas con resultados, unidades, metodología, ambientes, medios, bibliografía y evaluación.
2. **Coherencia académica**: evalúa consistencia entre resultados de aprendizaje, contenidos, cronograma, evaluación y evidencias.
3. **Aprobación bloqueante**: formaliza el flujo borrador → revisión → aprobación → ejecución → cierre, con bloqueo de expedientes aprobados.
4. **Informe ejecutivo institucional**: consolida riesgos, score, observaciones y recomendaciones.
5. **Exportación institucional estructurada**: genera ZIP maestro por período/programa/curso.
6. **Auditoría de expediente**: versiones, observaciones, workflow, auditoría general y bloqueo.

## Operación recomendada

Banco institucional → Expediente → Planeador FD-GC71 → Coherencia académica → Aprobación bloqueante → Cortes y evidencias → FD-GC72 → Informe ejecutivo → Exportación institucional.
