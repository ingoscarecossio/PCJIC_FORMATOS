# Manual de usuario

## 1. Inicio de sesión
Ingrese con usuario y contraseña. El administrador inicial es `admin / Admin123*`. Cambie la clave al primer ingreso.

## 2. Expediente académico
Cree un expediente por asignatura/grupo/periodo. Este registro permite trazabilidad, avance, estado y vínculo con evidencias.

Estados sugeridos:
- Planeación
- En ejecución
- Corte parcial
- Cierre final
- Archivado

## 3. Planeador superior
El planeador tiene tres pasos:

### Datos base
Diligencie identificación de la asignatura, docente, programa, créditos, HTP, HTI y textos académicos.

### Planeación y evaluación
Registre unidades, horas presenciales o sesiones, horario real de clase y evaluación concertada. La evaluación debe sumar 100%.

### Validar y descargar
Revise errores y alertas. Descargue:
- FD-GC71 en Word
- Plantilla Excel de evaluación
- Calendario ICS para Outlook/Google Calendar
- Paquete completo ZIP

## 4. Evidencias
Suba listados, evaluaciones, actas, asistencias y soportes por curso. Todo queda registrado con usuario y fecha.

## 5. FD-GC72
Use el módulo de informe académico para cargar listados y calificaciones. El sistema calcula matriculados, desertores, aprobados, reprobados y porcentajes.

## 6. Validador institucional
Antes de entregar, valide:
- Evaluación suma 100%.
- Existe hoja de calificaciones.
- Hay estudiantes.
- El expediente tiene sesiones, evaluación y evidencias.

## 7. Respaldo
El administrador puede descargar un ZIP con base SQLite y evidencias.
