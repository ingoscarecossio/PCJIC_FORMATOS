# Suite Inteligente FD-GC71 / FD-GC72

Esta versión transforma la aplicación en una plataforma académica documental con automatización operacional.

## Nuevos módulos

1. **Cargador inteligente de Excel**: detecta encabezados, sugiere columnas, normaliza documentos, nombres, estados y notas.
2. **Comparador de cortes**: cruza listado inicial, corte parcial y corte final para calcular desertores, nuevos, riesgo académico, aprobados y reprobados.
3. **Semáforo del expediente**: mide datos base, módulos, cronograma, evaluación, evidencias, observaciones y score institucional.
4. **Exportación masiva**: genera ZIP maestro con paquetes por curso e índice institucional.
5. **Hash y QR documental**: crea ficha JSON, SHA-256 y QR de verificación del expediente.
6. **Asistente académico**: genera borradores editables para FD-GC71 y FD-GC72.

## Uso recomendado

- Cargue listado inicial al iniciar el curso.
- Use el planeador para generar FD-GC71 y plantilla de evaluación.
- Cargue cortes parcial/final en el comparador.
- Revise semáforo y validaciones antes de cerrar.
- Exporte expediente con hash para trazabilidad.
